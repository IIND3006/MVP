# ACAFOXX

Sistema de gestión y análisis desarrollado para el proyecto PICE (Proyecto Intermedio Consultoría Empresarial) de Ingeniería Industrial en la Universidad de los Andes.

---
# Descripción del Proyecto

ACAFOXX es una plataforma diseñada para apoyar la gestión operativa y administrativa de laboratorios y recursos organizacionales mediante herramientas tecnológicas, visualización de información y automatización de procesos.
El proyecto fue desarrollado bajo la metodología de consultoría aplicada de PICE, donde el equipo actúa como consultor junior para resolver un reto real de una organización utilizando conocimientos de Ingeniería Industrial, analítica y desarrollo tecnológico. # Objetivo

Diseñar una solución funcional que permita:
- Centralizar información operativa
- Optimizar procesos administrativos
- Mejorar el control de inventario y recursos
- Facilitar la toma de decisiones mediante visualización de datos
- Incrementar la eficiencia y trazabilidad de los procesos

---

# Funcionalidades Principales

- Gestión de inventario
- Dashboard administrativo
- Visualización de métricas y KPIs
- Gestión de cursos/laboratorios
- Registro y consulta de información
- Interfaz web interactiva
- Organización de datos operativos
